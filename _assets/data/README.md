## Adding New Datasets

When adding new datasets:
1. Use descriptive filenames (e.g., `housing_prices.csv` not `data1.csv`)
2. Include a brief description in this README
3. Document the source and any licensing restrictions
4. Keep file sizes reasonable (< 1MB preferred for version control)

## External Data Sources

The book also uses datasets from R packages:
- `dslabs` - Various datasets for learning data science
- `ISLR` - Stock market data (Smarket)
- `HistData` - Historical datasets (GaltonFamilies)
- `modeldata` - ML datasets (mlc_churn)
- `ggplot2` - diamonds dataset
