# Changelog

All notable changes to this book will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [3.0.0] - 2025-12-24

### Changed
- Updated to Third Edition
- Updated to R 4.5.2 and RStudio 2025.09.2

## [2.0.0] - 2024-12-27

### Added
- English localization (fully translated from Spanish)
- GenAI chapter covering LLM fundamentals and API usage
- Ethics checklist appendix
- Real estate market analysis case study
- tidymodels-based machine learning workflows
- CITATION.cff for proper academic citations
- CONTRIBUTING.md for community guidelines

### Changed
- Updated to R 4.4.2 and RStudio 2024.12.0
- Migrated all machine learning code from `caret` to `tidymodels`
- Replaced `%>%` pipe with native `|>` throughout
- Modernized `dplyr` verbs (e.g., `across()` instead of `mutate_at()`)
- Reorganized chapter structure for better flow
- Enhanced README with badges and quick start guide

### Removed
- Spanish-only content (now English)
- Deprecated `caret` code (replaced with `tidymodels`)
- Unused temporary configuration files

## [1.0.0] - 2020-01-01

### Added
- Initial release in Spanish
- Chapters on R fundamentals, visualization, statistics, and machine learning
- Exercises with solutions
- Integration with bookdown for web publishing
