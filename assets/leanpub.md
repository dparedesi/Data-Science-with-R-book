# Book Details

**Title:**
Data Science with R

**Subtitle:**
Data Analysis and Prediction Algorithms with R - Third Edition

**About this book:**
Learning to make data-driven decisions is today a cross-industry requirement. In this book, you will find everything from the basics of data analysis to visualization, probability theory, inference, modeling, and machine learning. The content is designed to provide opportunities for new talents looking to train as Data Scientists. This Third Edition has been updated with modern tools like `tidymodels` and includes a new section on Generative AI.

**Teaser text:**
Discover how to become a Data Scientist at your own pace with updated content and real-world examples. From data analysis to prediction algorithms with machine learning. This Third Edition includes new chapters on Generative AI, Ethics, and modern Machine Learning workflows.

**Meta description:**
data science, R, machine learning, probability, modeling, artificial intelligence, tidymodels, generative AI, llm
